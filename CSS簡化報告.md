# TanaPOS V5 - CSS 簡化報告

## 概述
為了專注於功能開發，將複雜的 UI 風格系統簡化為基礎功能性樣式。

## 變更摘要

### 📉 **大幅簡化**
- **文件大小**: 從 862 行縮減至 152 行 (減少 82.4%)
- **複雜度**: 移除所有特效、動畫和複雜主題樣式
- **維護性**: 大幅提升，專注於功能而非視覺

### 🎯 **保留功能**
- ✅ 基本 CSS 重置和變數系統
- ✅ 所有 10 種主題的切換功能（但使用統一樣式）
- ✅ 基本組件樣式（按鈕、輸入框、卡片、模態框）
- ✅ 實用工具類別（flex、gap、margin、padding 等）
- ✅ UIStyleProvider 完整功能

### 🗑️ **移除內容**
- ❌ 複雜的主題特效（玻璃質感、新擬物、賽博朋克等）
- ❌ 動畫效果（故障效果、發光、彈跳等）
- ❌ 特殊字體和複雜色彩系統
- ❌ 大量的 CSS 變數和陰影系統
- ❌ 響應式特效調整

### 🔧 **新的 CSS 架構**

#### 基本變數
```css
:root {
  --pos-primary: #4f46e5;      
  --pos-secondary: #0d9488;    
  --pos-success: #16a34a;      
  --pos-warning: #facc15;      
  --pos-danger: #dc2626;       
  --color-foreground: #111827; 
  --color-background: #ffffff; 
  --color-border: #d1d5db;     
  --border-radius: 0.375rem;
}
```

#### 基本組件
- **按鈕**: 簡單的主色背景，基本懸停效果
- **輸入框**: 邊框樣式，焦點狀態
- **卡片**: 基本邊框和陰影
- **模態框**: 簡單的背景遮罩

#### 實用工具類別
- **佈局**: `.flex`, `.flex-col`
- **間距**: `.gap-2`, `.gap-4`, `.mb-2`, `.mb-4`, `.p-2`, `.p-4`
- **文字**: `.text-sm`, `.text-lg`, `.font-bold`, `.text-center`
