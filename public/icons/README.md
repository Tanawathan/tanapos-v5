## 圖示生成建議：
1. 使用 https://realfavicongenerator.net/ 生成所有尺寸
2. 或使用 https://www.pwabuilder.com/ 的圖示生成工具
3. 確保圖示有適當的邊距和清晰度

目前使用 favicon.svg 作為後備圖示。
